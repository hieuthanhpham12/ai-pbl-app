
📘 HƯỚNG DẪN SỬ DỤNG ỨNG DỤNG AI-PBL (Tiếng Việt)

1. Cài Python 3.9+ từ: https://www.python.org/downloads/
2. Mở Terminal hoặc CMD tại thư mục chứa file
3. Gõ lệnh sau để cài thư viện:
   pip install -r requirements.txt
4. Chạy ứng dụng bằng lệnh:
   streamlit run app_en.py
5. Truy cập tại: http://localhost:8501

📝 File CSV chứa thư viện gợi ý phải nằm cùng thư mục với file .py
